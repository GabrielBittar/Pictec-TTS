Uma extensão para Google Chrome que utiliza a API do Gemini 1.5 Flash para descrever imagens de forma didática, focando em acessibilidade e educação.

A Google Chrome extension that uses the Gemini 1.5 Flash API to describe images didactically, focusing on accessibility and education.

---


🇧🇷 Português (PT-BR)

🌟 Funcionalidades

Análise Didática: O prompt é otimizado para descrever formas geométricas, gráficos e diagramas.

Acessibilidade Total: Suporte para navegação via teclado (TAB) e leitores de tela (ARIA Live Regions).

Atalho de Teclado: Pressione Alt + I para analisar a imagem selecionada.

Menu de Contexto: Clique com o botão direito em qualquer imagem para acionar a IA.

Saída de Voz (TTS): A descrição é lida automaticamente pelo motor de voz do navegador.

🛠️ Instalação (Passo a Passo)

* Baixe o Projeto: Faça o download deste repositório (ou use git clone) e extraia o conteúdo em uma pasta de sua preferência.

* Modo Desenvolvedor:

     * Abra o Chrome e vá em chrome://extensions.

     * Ative o Modo do desenvolvedor (canto superior direito).

* Carregar Extensão:

     * Clique em Carregar sem compactar (Load unpacked).

     * Selecione a pasta principal do projeto (onde está o arquivo manifest.json).

* Chave da API:

     * Clique no ícone de "quebra-cabeça" na barra do Chrome e fixe a extensão.

     * Clique no ícone da extensão, insira sua chave do Google AI Studio e clique em Salvar.

🖥️ Como Usar

🖱️ Mouse: Clique com o botão direito em uma imagem > "Gemini: Descrever Imagem para Estudo".

⌨️ Teclado: Use TAB para navegar até uma imagem (o navegador anunciará que ela está pronta) e pressione Alt + I.


🇺🇸 English (EN-US)

🌟 Features

Didactic Analysis: Optimized prompt to describe geometric shapes, charts, and diagrams.

Full Accessibility: Keyboard navigation support (TAB) and Screen Reader compatibility (ARIA Live Regions).

Keyboard Shortcut: Press Alt + I to analyze the currently focused image.

Context Menu: Right-click any image to trigger the AI analysis.

Voice Output (TTS): Descriptions are automatically read aloud using the browser's speech engine.

🛠️ Installation (Step-by-Step)

* Download the Project: Download this repository (or use git clone) and extract it to a folder of your choice.

* Developer Mode:

     * Open Chrome and navigate to chrome://extensions.

     * Enable Developer mode (top right corner).

* Load Extension:

     * Click Load unpacked.

     * Select the project's main folder (the one containing the manifest.json file).

* API Key:

     * Click the "puzzle" icon in your Chrome toolbar and pin the extension.

     * Click the extension icon, enter your key from Google AI Studio, and click Save.

🖥️ How to Use

🖱️ Mouse: Right-click an image > "Gemini: Descrever Imagem para Estudo".

⌨️ Keyboard: Use TAB to navigate to an image (the browser will announce it is available) and press Alt + I.
